STANOWISKO: Frontend Developer (AngularJS)

Jeśli HTML i CSS nie mają dla Ciebie tajemnic, pojęcia takie jak one-page application i RWD są Ci dobrze znane, a JavaScript i AngularJS są Twoim żywiołem, to czytaj dalej.

Tagi: Angular, JavaScript, CSS, HTML, Front-End

Opis stanowiska

Obecnie poszukujemy osób na stanowisko: Front-End Developer (AngularJS), do którego zadań należeć będzie projektowanie architektury i implementacja aplikacji frontendowych z sektora finansowego.

Wymagania:

  • Bardzo dobra znajomość HTML/CSS3
  • Bardzo dobra znajomość JavaScript i jQuery
  • Znajomość i doświadczenie w pracy z AngularJS
  • Doświadczenie w Responsive Web Design oraz znajomość Bootstrap 3.0
  • Znajomość systemu kontroli wersji (GIT)

Mile widziane:

  • Znajomość platformy Heroku
  • Podstawowa znajomość NodeJS
  • Znajomość Bower, Grunt, Express.js, Underscore.js
  • Podstawowa znajomość Photoshopa
  • Znajomość metodyki Scrum / pracy w środowisku Agile

Oferujemy:

  • Elastyczne godziny pracy
  • Atrakcyjne wynagrodzenie i premie projektowe
  • Udział w ciekawych i zaawansowanych technologicznie projektach dla klientów z Polski i Europy
  • Dużą samodzielność w działaniu oraz możliwość realizacji własnych pomysłów w obrębie projektu
